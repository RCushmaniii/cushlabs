## Lessons / Surprises

- **Static Astro + server-side chatbot:** `output: 'static'` means NO API routes in the Astro site. All server-side logic must live in a Cloudflare Worker or separate service. This is the right constraint — don't change it for one feature.
- **CSP `frame-src 'none'` was blocking all iframes.** The existing security posture was strict (good), but needed updating to allow the demo worker iframe specifically. Any future third-party embed (Calendly, Loom, etc.) will need a CSP update in `vercel.json`.
- **`wrangler secret put` is interactive** — can't pipe from within Claude. Robert must run this manually. This is intentional Wrangler security behavior.
- **Guarantee "we" vs "I":** Robert's "I" voice preference (solopreneur positioning) applies to the guarantee copy too. Caught post-commit — flag for next session.
