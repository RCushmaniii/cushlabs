## Search Strategy Learnings

| Search Query | Result Quality | Why |
|-------------|---------------|-----|
| "Companies looking for AI consulting" | **Poor** — returns vendor content, not buyers | Vendor-intent query |
| "Hiring AI consultant / automation engineer" | **Strong** — surfaces companies with confirmed budget + need | Buyer-signal query via job boards |
| "Customer service automation for small business" | **Poor** — returns tool comparisons | Vendor-intent query |
| Tracing BPO client testimonials | **Strong** — named companies with verified pain points | Referral-chain prospecting |
| DTC brands with CX hiring signals | **Strong** — companies building CS function from scratch | Hiring gap = consulting opportunity |

**Best prospecting method so far:** Job board scanning for AI/automation/CX roles at companies in the 50-500 employee range. These companies have identified the need and allocated budget — you're not selling the problem, you're offering a faster solution.
